# Project Description

Lung cancer is one of the leading causes of cancer-related deaths worldwide. Early detection plays a crucial role in improving survival rates, but analyzing CT scan images manually can be time-consuming and requires expert radiologists.

This project presents an **AI-based lung cancer detection system** that analyzes **CT scan images of lungs** and predicts whether the lungs are **cancerous or normal** using a **Convolutional Neural Network (CNN)**.

The trained model is deployed through a **Streamlit web application**, allowing users to upload CT scan images and receive predictions instantly.

---

# Objectives

The main objectives of this project are:

- Develop a **deep learning model** to classify lung CT scan images.
- Detect whether the lungs are **cancerous or normal**.
- Apply **image preprocessing and CNN architecture** for medical image classification.
- Deploy the trained model as a **user-friendly web application**.

---

#  Dataset

The dataset used in this project is the **IQ-OTHNCCD Lung Cancer Dataset**.

The dataset contains CT scan images categorized into three classes:

- **Benign**
- **Malignant**
- **Normal**

For simplification, the problem was converted into **binary classification**:

| Original Classes | Final Class |
|------------------|-------------|
| Benign + Malignant | Cancer |
| Normal | Normal |

---

# ⚙️ Methodology

The project follows a standard **machine learning pipeline**.

## 1️ Data Preprocessing

Several preprocessing steps were applied to prepare the dataset for training:

- Image resizing to **128 × 128 pixels**
- Pixel normalization (0–255 → 0–1)
- Splitting dataset into:
  - **Training set (80%)**
  - **Testing set (20%)**
- Data augmentation techniques such as:
  - Image rotation
  - Flipping
  - Shifting

These steps help improve model performance and reduce overfitting.

---

## 2️ CNN Model Architecture

The deep learning model is built using **TensorFlow and Keras**.

The architecture includes:

- Convolutional Layers → Feature extraction
- MaxPooling Layers → Reduce spatial dimensions
- Flatten Layer → Convert feature maps to vector
- Dense Layers → Classification
- Dropout Layer → Prevent overfitting
- Sigmoid Activation → Binary classification output

Model configuration:

- **Loss Function:** Binary Cross Entropy  
- **Optimizer:** Adam  
- **Batch Size:** 32  
- **Input Size:** 128 × 128 × 3  

---

## 3️ Model Training

The model was trained using the training dataset and validated using the testing dataset.

Training involved multiple epochs where the model learned patterns in lung CT scan images to differentiate between **cancerous and normal lungs**.

---

## 4️ Model Evaluation

The performance of the model was evaluated using:

- **Accuracy**
- **Validation Accuracy**
- **Confusion Matrix**
- **Accuracy Graph**

These metrics help measure how well the model generalizes to unseen data.

---

#  Results

The trained model achieved **high classification accuracy** in distinguishing between cancerous and normal lung CT scans.

Outputs generated by the system include:

- Prediction result (**Cancer / Normal**)
- Confidence score
- Accuracy graph
- Confusion matrix

These results demonstrate that **deep learning can assist in early lung cancer detection** using medical imaging.
